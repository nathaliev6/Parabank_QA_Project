# ParaBank QA Testing Project

This repository contains a personal **Manual QA Testing Project** based on the ParaBank Demo online banking application.

The purpose of this project is to demonstrate practical QA skills through structured testing of core banking functionalities.

---

## Project Files

### `test_scenarios.csv`
High-level test scenarios covering key modules:

- Login
- Registration
- Accounts Overview
- Transfer Funds
- Bill Payment
- Profile Update
- Logout

### `test_cases.csv`
Detailed manual test cases including:

- Preconditions
- Description
- Test steps
- Test data
- Expected results
- Priority

### `bug_reports.csv`
Sample defect reports based on realistic banking system testing scenarios.

Examples:

- Invalid transfer amount
- Missing field validation
- Session handling after logout
- UI issues

### `test_plan.md`
Short test planning document describing scope and testing approach.
